# Source Notes

## Preserved decisions

- Single-face symbol dome rather than a true 3D sphere.
- Continuous right rotation.
- Ocean glyphs: `%` and `x`.
- Land glyph: `#`, same text size as ocean.
- Land is distinguished by color, alpha and dispersed position rather than larger glyphs.
- Four dispersed land masses on the visible face.
- Ocean symbols keep a subtle flicker.
- Edges keep soft blur and fade.
- Mouse only adds a slight facing bias; it does not steer rotation direction.
